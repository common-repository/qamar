# Open Font License (OFL)

## License Description

All fonts in this collection are licensed under the Open Font License (OFL). This license allows you to redistribute, modify,
and even use the fonts in commercial software.

For more information about the OFL license, please refer to the `OFL.txt` file included in the any font folder

## Goals of OFL

The OFL license aims to achieve some primary goals:

1. **Freedom to Use:** You can use these fonts for any purpose, including commercial projects, without any restrictions.

2. **Freedom to Share:** You can share these fonts with others.

3. **Freedom to Modify:** You can modify the fonts and distribute the modified versions.

(Read more about this license: https://openfontlicense.org)

## OFL File

The OFL.txt file contains the complete text of the OFL license. Please read this file,
and any questions or ambiguities you may have should be clarified by the information in this file.

## Note

Please adhere to all the rules and regulations of the OFL license when using these fonts.
As a suggestion, include license information in the documentation and README of your projects.

# Font Conversion

## Font Formats

While some of the fonts in this collection were originally in TTF (TrueType Font) format, we converted them to WOFF (Open Web Font Format)
and WOFF2 formats with the help of conversion tools to increase compatibility and performance on the web.

## Conversion Tool

The conversion from TTF to WOFF & WOFF2 was accomplished using the [transfonter] =>(https://transfonter.org/) 

& [webfont.yabe.land] =>(https://webfont.yabe.land/en/misc/convert-ttf-woff2/)

& [www.onlinewebfonts.com/] =>(https://www.onlinewebfonts.com/tools) & (https://convertio.co) & (https://cloudconvert.com/) utility.
